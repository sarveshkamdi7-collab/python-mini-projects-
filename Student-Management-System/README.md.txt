# Student Management System

A console-based Student Management System developed in Python to practice Object-Oriented Programming (OOP), file handling, exception handling, and data management.

## 📖 About

This project allows users to manage student records through a simple command-line interface. It demonstrates the implementation of core Python concepts by performing operations such as creating student records, validating input, storing data, and retrieving information from files.

## ✨ Features

- Add new student records
- Display student details
- Validate user input using custom exceptions
- Store student records in a file
- Read and display records from the file
- Object-Oriented Programming implementation
- Maintain multiple student objects using lists

## 🛠️ Technologies Used

- Python 3
